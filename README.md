## Noi dung Lab 2 da lam

- Xay dung navbar responsive co hamburger menu.
- Thay doi color scheme theo huong emerald/violet/orange.
- Ap dung dark mode bang class-based dark variant.
- Tao va su dung Client Components:
	- Counter
	- LikeButton
	- ThemeToggle
	- CopyButton
- Thuc hanh bo use client trong Counter de quan sat loi build, sau do khoi phuc.

## Huong dan chay

### 1) Chay bai JavaScript o root

```bash
node hello.js
```

### 2) Chay ung dung Next.js

```bash
cd ctk46b-cong-nghe-moi-lab01
npm install
npm run dev
```

Mo trinh duyet tai http://localhost:3000

## Kiem tra lint/build

```bash
cd ctk46b-cong-nghe-moi-lab01
npm run lint
npm run build
```
